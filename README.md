# Pi Prompt Title

Pi Prompt Title is an extension for [Pi](https://github.com/earendil-works/pi) that automatically gives a new session a concise, meaningful name based on its initial substantive prompt. It is intended to make session histories easier to scan and revisit without requiring users to name each session manually.

The project focuses on accomplishing that with negligible cost and minimal added latency. Rather than automatically spending another request on the active—and potentially expensive—session model, it will pursue a deliberately cost-conscious approach while targeting Pi's current extension API.
